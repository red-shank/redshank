#### Props

You can pass other props from [Box](/docs/components/box#apis) to the Center component.

| Attribute    | Type                  | Accepted values          | Description           | Default |
|--------------|-----------------------|--------------------------|-----------------------|---------|
| **children** | `string`  `ReactNode` | any                      | Component to center   |         |
| **size**     | `ContainerSize`       | `xs` `sm` `md` `lg` `xl` | Spacing padding added | `sm`    |
