### APIs

#### Carousel props
Api props extends from [ScrollView](https://reactnative.dev/docs/scrollview)

| Attribute           | Type                       | Accepted values                     | Description                                                                                   | Default |
|---------------------|----------------------------|-------------------------------------|-----------------------------------------------------------------------------------------------|---------|
| **children**        | `JSXElemnt` `JSXElement[]` | `React Element`                     | Accessory as a child for the component, we suggest passing Header, Content or Footer elements | -       |
| **scrollViewProps** | `ScrollViewProps`          | `Omit<ScrollViewProps, 'children'>` | Extra accessory for ScrollView element                                                        | -       |
| **showScroll**      | `boolean`                  | `true` `false`                      | Show or Hidden scroll bars                                                                    | `false` |
| **widthChild**      | `boolean`                  | `true` `false`                      | Subtract 60px to the element total width                                                      | `false` |
