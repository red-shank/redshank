#### Props

Pass all props to the ScrollView component from React Native.
More info about `SxProps` in [The sx prop](/docs/the-sx-prop)

| Attribute              | Type     | Accepted values | Description        | Default |
|------------------------|----------|-----------------|--------------------|---------|
| **sx**                 | `object` | `SxProps`       | Root sx props      |         |
| **contentContainerSx** | `object` | `SxProps`       | Container sx props |         |
