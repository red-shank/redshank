### When To Use
When requiring users to interact with the application, but without jumping to a new page and interrupting the user's workflow, you can use Modal to create a new floating layer over the current page to get user feedback or display information. Additionally, if you need show a simple confirmation dialog, you can use [Pop Confirm](/docs/components/pop-confirm), and so on.

### APIs

<Spacer y={2} />

#### Modal
Api props extends from [Modal](https://reactnative.dev/docs/modal)

| Attribute         | Type                            | Accepted values                               | Description                                         | Default                                         |
|-------------------|---------------------------------|-----------------------------------------------|-----------------------------------------------------|-------------------------------------------------|
| **animationType** | `Enum`                          | `fade`                                        | `slide`                                             | `none`                                          | Change background color of the input                | `inputColor`                            |
| **children**      | `ReactNode`                     | `Component` `JSX.ELement`                     | Content for show in the modal                       | -                                               |
| **closable**      | `boolean`                       | `true` `false`                                | Show or hidden close button                         | `false`                                         |
| **extra**         | `ReactNode` `JSX.Element`       | `Component`                                   | Extra content in bottom modal                       | -                                               |
| **fullScreen**    | `boolean`                       | `true` `false`                                | Show content in full screen                         | `false`                                         |
| **hiddenBar**     | `boolean`                       | `true` `false`                                | Show or hidden statusbar in mobiles                 | `false`                                         |
| **statusHeight**  | `Number`                        | `Int`                                         | Status height for not render content here           | `automatic`                                     |
| **maskClosable**  | `boolean`                       | `true` `false`                                | Allow close modal when press outside modal          | `false`                                         |
| **maskComponent** | `Component` `FunctionComponent` | `JSX.Element` `Component`                     | Change component mask modal                         | `Box` if `maskClosable` then `TouchableOpacity` |
| **onClose**       | `function`                      | `() => void`                                  | Required function for close modal                   | -                                               |
| **position**      | `PositionType`                  | `top` `center` `bottom`                       | Position of the modal, when `fullScreen` is `false` | `center`                                        |
| **scrollable**    | `boolean`                       | `true` `false`                                | Can scroll in modal content                         | `false`                                         |
| **visible**       | `boolean`                       | `true` `false`                                | Required controller show or hidden modal            | -                                               |
| **sx**            | `object`                        | [Sx customization](#sx-customization)         | Sx props for components                             |                                                 |
| **styles**        | `object`                        | [Styles customization](#styles-customization) | Styles props for components                         |                                                 |

### Sx Customization

More info about `SxProps` in [The sx prop](/docs/the-sx-prop)

```ts
type Sx = {
  root?: SxProps;
  touchable?: SxProps;
  thumb?: SxProps;
  toggle?: SxProps;
  helperText?: SxProps;
};
```

#### useModal
Api for show or hidden modal with hook.

##### Basic use
```tsx
const [visible, toggleModal] = Modal.useModal();
```

##### Completed use
```tsx
const [
  visible,
  toggleModal,
  { onVisible, onHidden }
] = Modal.useModal();
```
