### APIs

<Spacer y={2}/>

#### Props

| Attribute           | Type                    | Accepted values                               | Description                                        | Default      |
|---------------------|-------------------------|-----------------------------------------------|----------------------------------------------------|--------------|
| **backgroundColor** | `ThemeColor` `string`   | [ThemeColor](/docs/default-theme)             | Background calendar color                          | `background` |
| **cancelText**      | `string`                | `Text`                                        | Cancel text for button footer                      | `Cancel`     |
| **defaultSelected** | `string`                | `YYYY-MM-DD`                                  | Default selected date in calendar                  | `now`        |
| **disabled**        | `boolean`               | `true`, `false`                               | Disabled the calendar                              |              |
| **locale**          | `string` `dayjs.locale` | `en`, `es`, `fr`, `etc`                       | Valid locale for dayjs                             | `en`         |
| **max**             | `string`                | `YYYY-MM-DD`                                  | Valid string date for max posible selected date    |              |
| **min**             | `string`                | `YYYY-MM-DD`                                  | Valid string date for min posible selected date    |              |
| **okText**          | `string`                | `Text`                                        | Ok text for button footer                          | `Ok`         |
| **onCancel**        | `function`              | `() => void`                                  | Function for Cancel button in footer               |              |
| **onChange**        | `function`              | `(date: string) => void`                      | Function for  ok button in footer                  |              |
| **onSelected**      | `function`              | `(date: string) => void`                      | Trigger when user select a day                     |              |
| **selected**        | `string`                | `YYYY-MM-DD`                                  | Selected date in calendar for controller component |              |
| **sx**              | `object`                | [Sx customization](#sx-customization)         | Sx props for components                            |              |
| **styles**          | `object`                | [Styles customization](#styles-customization) | Styles props for components                        |              |

<Spacer y={2}/>

### Sx Customization

More info about `SxProps` in [The sx prop](/docs/the-sx-prop)

```ts
type Sx = {
  container?: SxProps;
  daysNameWeekItem?: SxProps;
  daysNameWeekWrapper?: SxProps;
  daysOfWeekWrapper?: SxProps;
  daysOfWeekItem?: SxProps;
  daySelected?: SxProps;
  dayNow?: SxProps;
  layout?: SxProps;
};
```
