#### Title Props
More props in [TextProps](/docs/text#apis)

| Attribute        | Type                  | Accepted values                   | Description                 | Default     |
|------------------|-----------------------|-----------------------------------|-----------------------------|-------------|
| **color**        | `ThemeColor` `string` | [ThemeColor](/docs/default-theme) | Change icon color           | `text`      |
| **children**     | `string`  `ReactNode` | `any`                             | Content to show             | `undefined` |
| **level**        | `number`              | `1` `2` `3` `4` `5`               | Level to title, change size | `1`         |
| **align**        | `string`              | [Align](#align)                   | Align title text            | `left`      |
| **transform**    | `string`              | [Transform](#transform)           | Transform title text        | `none`      |
| **marginBottom** | `number`              | `number`                          | Margin bottom to element    | `undefined` |
