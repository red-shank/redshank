### When to use

Tabs are implemented using a collection of related components:

* `<Tab/>` - the container that houses the tabs. Responsible for handling focus and keyboard navigation between tabs.
* `children` - the tab element itself. Clicking on a tab displays its corresponding panel.

### APIs

<Spacer y={2}/>

#### Props

Pass all props to the ScrollView component from React Native.
More info about `SxProps` in [The sx prop](/docs/the-sx-prop)

| Attribute              | Type                | Accepted values                                       | Description                        | Default |
|------------------------|---------------------|-------------------------------------------------------|------------------------------------|---------|
| **items**              | `array`             | [TabItemProps](#tab-item-props)                       | Array of element to render in tabs |         |
| **activeKey**          | `NumberStringValue` | `string`, `number`                                    | Controller tabs active tab         |         |
| **defaultActiveKey**   | `NumberStringValue` | `string`, `number`                                    | Default active tab                 |         |
| **scrollable**         | `boolean`           | `true`, `false`                                       | Enabled scroll horizontal          | `false` |
| **onChange**           | `function`          | (item: [TabItemProps](#tab-item-props)) => `void`     | On callback when item is pressed   |         |
| **labelProps**         | `object`            | [Text Props](/docs/components/text)                   | Label props for item               |         |
| **variant**            | `Enum`              | [TabsVariant](#tabs-variant)                          | Variant for UI                     |         |
| **backgroundColors**   | `object`            | { background: `ColorType`, borderColor: `ColorType` } | Props for custom item tabs         |         |
| **sx**                 | `object`            | [Sx customization](#sx-customization)                 | Sx props to elements               |         |
| **contentContainerSx** | `object`            | [Styles customization](#styles-customization)         | Styles props to element            |         |

<Spacer y={2}/>

### Sx Customization
More info about `SxProps` in [The sx prop](/docs/the-sx-prop)


```ts
type Sx = {
  root?: SxProps;
  tab_container?: SxProps;
  tab_scroll_view?: SxProps;
  content?: SxProps;
  item?: SxProps;
}
```
